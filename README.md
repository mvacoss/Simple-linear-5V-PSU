# Simple-linear-5V-PSU
- Simple linear PSU design with LM7805
- Without watermark, without useless names
- It is double sided, but you can also export only single sided layout (B_Cu), so it is ideal for DIY etching
  
- I included already exported PDF of the [B_Cu](https://github.com/user-attachments/files/26326793/Etch.pdf) too, so you don't have to open KiCad at all! But remember, the side with the ink on it goes directly to the copper layer of your PCB

## Specs
- ~6V - ~10V IN
- +5V OUT
- MAX 1A (with suitable heatsink)
- Onboard LED indication (Lights up when the output is +5V)
